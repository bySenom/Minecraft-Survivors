Minecraft Survivors

A Vampire‑Survivors‑style minigame for Paper/Spigot — continuous spawns, classes, glyphs & loot.

Repository README CI Status Wiki API

Kurzfassung

Minecraft Survivors ist ein erweiterbares Minigame-Plugin, das das Spielgefühl von Vampire Survivors in Minecraft bringt: kontinuierliche Gegner‑Spawns, Klassen mit einzigartigen Fähigkeiten, Level‑Ups mit Raritäten und ein Party/Shop/Stats System.

Highlights

  • Klassen

    Shaman, Pyromancer, Ranger, Paladin — jede mit eigenen Fähigkeiten.

  • Continuous Spawns

    Gegner wie in Vampire Survivors, dynamisch skaliert mit Enrage und Elite‑Spawns.

  • Glyphen & Loot

    Sockelbare Glyphen, Level‑Ups mit Raritäten (Common / Rare / Epic).

  • Party & Ready Flow

    Party‑System mit Leader‑Votes und Ready‑Checks, in‑GUI Einladungen.

  • Stats & Meter

    DPS/HPS Meter, Anzeige per ActionBar/BossBar/Scoreboard.

  • Shop & Progression

    Daily Offers, Limits, Meta‑Progression.

Install

  1. Build: ./gradlew.bat clean build (Windows) or ./gradlew clean build
  2. Copy build/libs/*.jar to <paper-server>/plugins/
  3. Start server (Paper 1.21.4 recommended)

Schnellbefehle

/msmenu – Hauptmenü
/msstart – Admin: Spiel starten
/party – Party‑Befehle
