Kurzer Einstieg

- Build: `./gradlew clean build`
- Kopiere die erzeugte JAR in `<paper-server>/plugins/`
- Starte den Server (Paper 1.21.4 empfohlen)
- Öffne In‑Game das Menü mit `/msmenu`

Wenn du Inhalte ergänzen möchtest, erstelle einen PR oder öffne ein Issue im Repo.
