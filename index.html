Get started Wiki API View on GitHub

Kurzfassung

Minecraft Survivors ist ein leicht erweiterbares Plugin, das das Spielgefühl von Vampire Survivors in Minecraft bringt: kontinuierliche Gegner‑Spawns, Klassen mit einzigartigen Fähigkeiten, Level‑Ups mit Raritäten und ein Party/Shop/Stats System.

Install & run How to install
Quick Install (Windows) 
cd C:\\Users\\SashaW\\IdeaProjects\\MinecraftSurvivors
.\\gradlew.bat clean build
copy build\\libs\\*.jar <your-paper-server>\\plugins\\
Paper 1.21.4 empfohlen — starte dann den Server und nutze /msmenu

Highlights

Klassen

Shaman, Pyromancer, Ranger, Paladin — jede mit eigenen Fähigkeiten und Progression.

Continuous Spawns

Vampire Survivors‑like Spawns mit Enrage und Elite‑Mobs.

Glyphen & Loot

Pick‑ups, socketing und automatische Ability‑Levelups bei vollen Sockeln.

Party & Ready Flow

Party‑Leader start Ready‑Vote, gemeinsame Klassenwahl und synchroner Start.

Stats & Meter

DPS/HPS Meter (ActionBar/BossBar/Scoreboard) mit Sliding Window.

Shop & Meta

Daily Offers, permanent upgrades & NPC Shop.
